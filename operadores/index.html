Operadores aritméticos
OPERADOR FUNCION EJEMPLO
= Asignación :
Permite dar un valor a una variable o constante 		var variable = "Valor";
let local = true;
const constante = 10;
+ Suma / Concatenación :
Permite sumar dos valores numéricos o concatenar (unir) dos cadenas de texto (o una cadena con un valor) 		var suma = 5 + 10; // 15

var cadena = "Unión de "+3+" valores";
// "Unión de 3 valores"
- Resta :
Permite restar dos valores numéricos 		var resta = 20 - 3; // 17
* Multiplicación :
Realiza multiplicaciones entre valores numéricos 		var multiplicacion = 8 * 7; // 56
/ División :
Realiza una división y regresa el resultado de la misma 		var division = 20 / 4; //5 var division = 14 / 3; //3
% Módulo :
Realiza una división, pero, regresa el sobrante de la misma, es útil para saber si un número es divisible entre otro 		var modulo = 20 % 4; //0
var modulo = 14 % 3; //2
/*En este caso 14/3 = 3 y tiene un sobrante de 2*/
** Potencia :
Permite elevar un valor a una potencia dada, donde el valor a la derecha se eleva a la potencia en la izquierda 		var potencia = 3 ** 2; // 9
++ Incremento :
Realiza un incremento paulatino de una unidad a el valor que corresponda 		var a = 5;
console.log(a++); // 6
-- Decremento :
Disminuye en una unidad el valor al cual sea asignado 		var c = 0;
console.log(c--); // -1
+= Suma y asignación :
Retorna la suma de los valores que conforman la operación y la asigna a la variable correspontiente 		var a = 5;
var b = 3;
b+=a;
consolo.log(b); // 8
-= Resta y asignación :
Realiza la diferencia entre los dos valores y lo asgina a la variable que esta indicada 		var a = 5;
var b = 3;
b-=a;
consolo.log(b); // -2
*= Multiplicación y asignación :
Realiza el producto y lo asigna a la variable 		var a = 5;
var b = 3;
b*=a;
consolo.log(b); // 15
/= División y asignación :
Realiza la división y asigna el cociente a la variable 		var a = 5;
var b = 3;
b/=a;
consolo.log(b); // 0.6
Operadores relacionales
OPERADOR FUNCION EJEMPLO
> Mayor que :
Permite validar si un valor a es mayor que un valor b 		var a = 5;
var b = 3;
console.log( a > b ); //true
console.log( a > 5 ); //false
< Menor que :
Permite validar si un valor a es menor que un valor b 		var a = 5;
var b = 3;
console.log( b < a ); //true
console.log( b < 3 ); //false
>= Mayor o igual a :
Permite validar si un valor a es mayor o igual que un valor b 		var a = 5;
var b = 3;
console.log( a >= b ); //true
console.log( a >= 5 ); //true
<= Menor o igual a :
Permite validar si un valor a es menor o igual que un valor b 		var a = 5;
var b = 3;
console.log( b <= a ); //true
console.log( b <= 3 ); //true
== Igual a :
Permite validar si dos valores (a, b) son iguales, dejando de lado el tipo de dato, por lo que un valor numerico y una cadena son tratados como iguales 		var a = 5;
var b = 3;
console.log( b == a ); //false
console.log( a == 5 ); //true
console.log( a == "5" ); //true
=== Identico que a : (Igualdad estricta)
Permite validar si dos valores son iguales en valor y tipo de dato, haciendo clara distinción entre una cadena y un valor numérica 		var a = 5;
var b = 3;
console.log( b === a ); //false
console.log( a === 5 ); //true
console.log( a === "5" ); //false
!= Diferente de :
Permite validar si dos valores (a, b), son diferentes en valor, pero sin distinción de tipo 		var a = 5;
var b = 3;
console.log( b != a ); //true
console.log( a != 5 ); //false
console.log( a != "5" ); //false
!== Distinto a : (Deferencia estricta )
Permite validar si dos valores (a, b), son diferentes en valor, pero haciendo distinción de tipo de dato 		var a = 5;
var b = 3;
console.log( b !== a ); //true
console.log( a !== 5 ); //false
console.log( a !== "5" ); //true
Operadores lógicos
OPERADOR FUNCION EJEMPLO
&& And (Y) :
Relaciona dos condiciones, haciendo obligatorio que ambas se cumplan para que se considere verdadero, si una falla, se considera falso 		var a = 5;
var b = 10;
var c = 12;
console.log( a < b && a < c ); //true
console.log( a < b && b < c ); //true
console.log( a < b && a > c ); //false
|| OR (O) :
Relaciona dos condicionales, tales que cumpliendose almenos una de ellas, la condición se valida como verdadera, si ambas fallan se considera falso 		var a = 5;
var b = 10;
var c = 12;
console.log( a < b || a < c ); //true
console.log( a < b || b < c ); //true
console.log( a < b || a > c ); //true
! Not (Negación) :
Cambia el valor de una condición de verdadero a falso o falso a verdadero 		var a = 5;
var b = 10;
var c = 12;
console.log( ! a < b ); //false
console.log( ! b < c ); //false
console.log( ! c < a ); //true
? Operador ternario :
Permite realizar condiciones en una sola línea, formado de 3 partes obligatorias condicion ? true : false, donde cada valor se va a retornar según el resultado de la condición 		var a = 5;
var b = 10;
var c = 12;
console.log( a < b ? "A es menor" : "B es menor" );
//"A es menor"

console.log( a > c ? "A es mayor" : "C es mayor" );
//"C es mayor"
XOR
(No existe) 		La condición XOR permite invertir la condición or, es decir en caso de que ambas son validas se retorna false, cuando ambas son falsas se devuelve false, pero si una es valida y otra falsa se retorna true var a = 5;
var b = 10;
var c = 5;
console.log( ! ( a < b && a < c ) ); //true
console.log( ! ( a < b && b < c ) ); //true
console.log( ! ( a > b && a > c ) ); //false
