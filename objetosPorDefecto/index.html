LOS OBJETOS DENTRO DE JAVASCRIPT

Number

Es un objeto que nos permite manipular valores y validar características de números o variables que contengan números, esta clase posee constantes, metodos de validación y conversión.

Constantes

MAX_VALUE
Contiene el límite de valores positivos (R) que es capaz de representar JS
MIN_VALUE
Contiene el límite de valores negativos (R) que es capaz de representar JS
MAX_SAFE_INTEGER
Contiene el límite de valores enteros positivos (N) que es capaz de representar JS
MIN_SAFE_INTEGER
Contiene el límite de valores negativos enteros (Z) que es capaz de representar JS
EPSILON
Usado para obtener el número más pequeño próximo a 0
POSITIVE_INFINITY
Almacena el resultado de obtener un valor tan grande que JS no puede mostrarlo con sus valores soportados, por ejemplo:
100**900 ; //Infinity
NEGATIVE_INFINITY
Es el valor que indica un número muy pequeño cuando no es posible representarlo dentro de los estándares de JS, por ejemplo:
(-500)**1001 ; //-Infinity
NaN
Notación de no es un número (Not a Number), es el valor que nos retorna por defecto cuando intentamos usar valor inválido dentro de una operación, por ejemplo multiplicar una cadena de texto por un valor:
"hola"*5 ; //NaN

Funciones de validación

  • isInteger( number )
  • isSafeInteger( number )
  • isFinite( number )
  • isNaN( number )

Funciones de conversión

  • toString()
  • parseInt( number, base )
  • parseFloat( string )
